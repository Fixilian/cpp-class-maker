# C/C++ Class Maker

Create C/C++ header, source and test files in one click — with automatic CMake and Git integration.

---

## Features

- **Generate files instantly**
  - Creates header (`.h`, `.hpp`) and source (`.cpp`) files for new classes.
  - Generates optional unit test files.
- **CMake integration**
  - Automatically adds new files to your existing `CMakeLists.txt`.
- **Git integration**
  - Adds generated files to your repository and stage area.
- **Configurable templates**
  - Customize file structure and naming patterns.
- **Context menu support**
  - Right-click a folder or file and create new class elements directly.

---

## Usage

1. Open a C/C++ project with CMake.
2. Right-click on a folder or file in the Explorer.
3. Choose **“Create C++ Class”** (or use the Command Palette).
4. Enter the class name.
5. Your header, source, and test files will be generated automatically and added to CMake and Git.
